# Созданные документы

Созданы документы для проекта AstroPhoto:

- `docs/AstroPhoto_User_Guide_RU.docx`
- `docs/AstroPhoto_Algorithm_And_App_Logic_RU.docx`

Файлы `.docx` собраны через `python-docx`.

Также созданы markdown-версии:

- `docs/AstroPhoto_User_Guide_RU.md`
- `docs/AstroPhoto_Algorithm_And_App_Logic_RU.md`

Как обновлять позже:

1. Сначала обновить соответствующий `.md` файл.
2. Пересобрать `.docx` из markdown-источника.
3. Открыть `.docx` в Word или LibreOffice и быстро проверить страницы глазами.
4. Если в приложении появились новые функции, добавлять их только после проверки по текущему коду проекта.
